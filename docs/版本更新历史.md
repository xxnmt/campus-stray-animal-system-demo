# 版本更新历史

## v0.0.7 (体验版) - 2026-06-08

### 修改内容
1. **修复头像选择缓存问题**
   - 文件：`miniprogram/components/modifyUserInfo/modifyUserInfo.wxml`、`miniprogram/components/modifyUserInfo/modifyUserInfo.js`、`miniprogram/components/modifyUserInfo/modifyUserInfo.wxss`
   - 使用 `wx.chooseMedia` API 替代 `button open-type="chooseAvatar"`，避免微信小程序的头像缓存机制
   - 支持从相册和相机选择图片
   - 添加 `display: flex` 和 `overflow: hidden` 样式修复

2. **修复云存储上传问题**
   - 文件：`miniprogram/components/modifyUserInfo/modifyUserInfo.js`
   - 在头像上传路径中添加时间戳：`/user/avatar/${openid}_${timestamp}.${ext}`
   - 确保每次上传都是新文件，避免 EMAS 缓存返回旧文件信息

3. **修复全局头像显示问题**
   - 文件：`miniprogram/pages/genealogy/commentBoard/commentBoard.js`、`miniprogram/pages/genealogy/feedbackDetail/feedbackDetail.js`、`miniprogram/components/photoRank/photoRank.js`、`miniprogram/pages/followFeed/followFeed.js`、`miniprogram/pages/manage/managers/managers.js`
   - 为所有显示用户头像的页面添加 `userInfoUpdated` 事件监听
   - 修复事件监听绑定问题，保存 `boundOnUserInfoUpdated` 引用
   - 在 `onUnload` 生命周期中移除事件监听，防止内存泄漏

4. **优化缓存清理机制**
   - 文件：`miniprogram/components/modifyUserInfo/modifyUserInfo.js`
   - 先清理缓存，再延迟 100ms 发布更新事件
   - 确保其他页面不会从缓存中获取旧数据

5. **移除冗余逻辑**
   - 文件：`miniprogram/utils/common.js`、`miniprogram/utils/user.js`、`functionsEMAS/unionOp/userOp.js`、`miniprogram/pages/info/userInfo/userInfo.js`
   - 移除所有多余的 URL 清理逻辑
   - 恢复简洁的上传流程
